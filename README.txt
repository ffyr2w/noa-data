Test data for ffyr2w/noa
This is meant to be an aritfact repository. The VCS is not used.

Usage:
    - Clone this repository: e.g. `git clone --depth=1 https://github.com/ffyr2w/noa-data.git`.
    - Go to the repository and run with bash: `./GenerateData.sh` to generate the test data.
    - Set the environmental variable NOA_TEST_DATA to the path of the repository.
    - Run noa_test.
